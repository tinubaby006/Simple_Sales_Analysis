# 📊 Sales Analysis Web App

A Flask-based web application that analyzes sales data and generates insights with visual charts.

## 🚀 Features

* Secure login system
* Input multiple products, quantity, and price
* Calculates total revenue
* Identifies best-selling product(s)
* Generates pie chart visualization

## 🛠 Tech Stack

* Python (Flask)
* HTML/CSS
* Matplotlib

## ▶️ How to Run

```bash
pip install -r requirements.txt
python app.py
```

Then open:

```
http://127.0.0.1:5000
```
